En esta preentrega presento un módulo de carga de stock para un tienda virtual de indumentaria.
Donde se procede a cargar:
cod del producto,
tipo de producto,
talle,
color,
precio unitario,
cantidad.

El el alcance de este módulo es:

carga de producto,
control de producto cargado,
modificacion de producto cargado,
eliminacion de producto cargado.


Nota#1: Para la salida de los productos cargados se eligio de esa forma debido que visualmente es más aceptable desde el alert(), ya que la idea original era mostrarla con un encabezado y los productos listados un debajo del otro, pero la salida quedaba desprolija.
